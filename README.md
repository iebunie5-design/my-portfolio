# ENFP Developer Portfolio

> ENFP 개발자의 미니멀 포트폴리오

## 기술 스택

- **Next.js 15** - React 프레임워크
- **TypeScript** - 타입 안전
- **Tailwind CSS** - 스타일링
- **App Router** - 라우팅

## 시작하기

```bash
npm install
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

## 커스터마이징

`mocks/profile.json`에서 이름, 기술 스택, 프로젝트 등을 수정하세요.
